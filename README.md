# Free2B

Free2B is a mobile app designed to make arts and culture more accessible by helping people discover free events and experiences in their city.

## Overview

Free2B was created to make it easier for people to find free local events in one place. The project reflects my interest in building technology that improves access to community experiences and cultural resources.

## My Role

I led Free2B from concept to execution. My responsibilities included:

- Defining the product vision and core features
- Coordinating with developers during implementation
- Managing timelines and progress across the project
- Testing functionality and identifying issues
- Shaping the overall user experience and direction of the app

## Features

- Discover free local events
- Explore arts, culture, and community experiences
- Mobile-first, accessible event discovery

## Tech Stack

- Flutter
- Firebase
